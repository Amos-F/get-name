# DeepSeek 智能起名助手

一个基于 Vue 3 + TypeScript + Vite 开发的智能起名应用，集成了 DeepSeek AI 接口，为用户提供专业的起名服务。

## 功能特点

- 💡 智能起名：根据用户需求提供个性化名字推荐
- 📚 文化解析：详细解释名字的含义和文化内涵
- 🎵 音律分析：确保名字音律和谐
- 💬 实时对话：即时响应用户的起名需求
- 📱 响应式设计：完美支持移动端和桌面端

## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- TypeScript - JavaScript 的超集
- Vite - 下一代前端构建工具
- DeepSeek API - AI 对话接口
